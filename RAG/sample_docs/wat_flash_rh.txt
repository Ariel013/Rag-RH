8. QUESTIONS / RÉPONSES RH
================================================================

Q : "Comment faire une demande d'avance sur salaire ?"
R : Via le Helpdesk > "Demande d'avance sur salaire". Maximum 50% du salaire net, une fois par trimestre. Traitement sous 48h ouvrables.

Q : "J'ai un problème avec mon ordinateur, que faire ?"
R : Créer un ticket sur le Helpdesk > "Support IT". En cas d'urgence, contacter M. Moussa Traoré sur Slack (@moussa.traore).

Q : "Comment savoir si ma demande de congé a été validée ?"
R : Vous recevez un email de confirmation de la RH dès validation. Si vous n'avez pas de réponse sous 48h, relancez jb.koffi@aeig.africa.

Q : "Peut-on faire du télétravail sans l'accord du manager ?"
R : Non. Tout jour de télétravail nécessite un accord préalable et écrit du manager via le Helpdesk. Sans validation, l'absence sera considérée comme injustifiée.

Vous avez une question RH ? Envoyez-la à : jb.koffi@aeig.africa
Elle pourra apparaître dans le prochain Flash RH (anonymisée si vous le souhaitez) !
